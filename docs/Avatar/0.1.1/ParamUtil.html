(static) GetParamCode(state, parameter_name)

Source:
  • ParamUtil.js, line 22

GetParamCode is used to get the current code value of a given string response parameter where the string response parameter has an associated list of selectable items.

Parameters:
Name Type Description
state Object

The Reportal scripting state object.
parameter_name String

The name of the string response parameter to get the value from.
Returns:
  • the string code value of the given parameter. If the parameter does not have a string code value null is returned.
